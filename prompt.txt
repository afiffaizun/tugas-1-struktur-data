Buatkan program Java sederhana untuk menyimpan data mahasiswa (NIM dan Nama) menggunakan array statis dengan kapasitas 10.
Jangan pakai ArrayList atau library lain, cukup array biasa.
Gunakan variabel count untuk menyimpan jumlah data yang terisi.

#Tambahkan menu :
- Insert at beginning
- Insert at given position
- Insert at end
- Delete from beginning
- Delete given position
- Delete from end
- Delete first occurrence
- Show data
- Exit
Program terus berjalan sampai user memilih Exit.

# Insert at beginning
Buat fungsi insert at beginning.
Data baru dimasukkan ke indeks 0, data lama digeser ke kanan.
Jangan lupa cek kalau array sudah penuh dan tambahkan count setelah insert.


#Insert at end
Buat fungsi insert at end.
Data langsung dimasukkan ke indeks count.
Tambahkan validasi jika array penuh.

#Insert at given position
Buat fungsi insert at given position.
Posisi yang dimasukkan user sesuai indeks array (0 sampai count).
Data setelah posisi tersebut digeser ke kanan.
Tambahkan validasi posisi dan array penuh.


#Delete from beginning
Buat fungsi delete from beginning.
Data di indeks 0 dihapus, data lain digeser ke kiri.
Kurangi count setelah delete.


#Delete from end
Buat fungsi delete from end.
Cukup kurangi count satu.
Pastikan data tidak kosong sebelum delete.


#Delete at given position
Buat fungsi delete at given position.
Posisi sesuai indeks array.
Data setelah posisi digeser ke kiri dan count dikurangi.


#Delete first occurrence
Buat fungsi delete first occurrence berdasarkan NIM.
Cari NIM dari awal array, hapus data pertama yang ketemu, lalu geser data.


#Show data
Buat fungsi show data untuk menampilkan semua data mahasiswa.
Tampilkan indeks array, NIM, dan Nama.
Jika data kosong, tampilkan pesan yang sesuai.


Rapikan kode agar mudah dibaca pemula.
Gunakan nama variabel yang jelas dan komentar singkat di setiap fungsi.
Jangan gunakan konsep yang terlalu advance.
